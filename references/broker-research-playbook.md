# 券商研究获取策略

当用户需要券商、银行、策略、行业、宏观或分析师研究时使用本文。

## 目标

找到最高质量的可公开访问版本，保持来源合法性和访问标签清晰。

## 可接受的券商研究类型

公开且通常可用的：
- 年度展望合集
- 行业展望
- 主题策略报告
- 宏观展望
- 会议演示材料
- 行业入门读本
- 公开洞察和思想领导力 PDF
- 策略年报

直接跳过的：
- 需要订阅、客户端登录或付费墙的内容
- 公司首次覆盖报告
- 盈利预览/回顾报告
- 估值更新
- 订阅者专用行业报告
- 分析师门户下载

### 高风险来源（不使用）

- 无来源追溯的文件托管平台镜像
- 论坛或博客上的转载
- 被剥离了出版方上下文的 PDF

## 搜索模式

### 1. 确定机构

规范化：
- 机构中文名称
- 机构英文名称（如有）
- 研究部门名称（如有区别）

### 2. 先搜索官方公开平台

查找：
- 洞察/研究页面
- 思想领导力栏目
- 会议/活动页面
- 年度展望合集

### 3. 按文档类别搜索

将机构名与以下组合：
- `outlook` / `展望`
- `strategy` / `策略`
- `sector` / `行业`
- `macro` / `宏观`
- `primer` / `入门`
- `conference` / `会议`
- `pdf`
- `行业深度`
- `策略年度`
- `晨会纪要`
- `专题报告`
- `深度报告`

### 4. 验证访问状态

分配以下标签之一：
- `official PDF`
- `official page`
- `wechat official account`
- `secondary mirror`

不要模糊这些类别。付费内容直接跳过，不纳入。

券商研究通过微信公众号渠道获取时，标注为 `wechat official account`，同时注明原始券商机构名称。

## 每项来源的输出字段

- 机构名称
- 文档标题
- 分析师、部门或研究团队（如有）
- 发布日期
- 主题或行业
- 访问标签
- 官方页面 URL
- 直接 PDF URL（如有）

## 决策规则

### 当只有二次转载的 PDF 存在时

- 仅当该主题的信息获取已被严重阻塞且完整性优先于纯净度时保留
- 标注为 `secondary mirror`
- 说明未找到原始公开版本

### 当出现多个近似重复的报告时

优先：
1. 机构旗舰报告优于简短评论
2. 用户时间窗口内的最新版本
3. 更广泛的行业或主题报告优于公司特定的维持性评论（除非用户明确要求公司覆盖）

### 中国市场特有规则

积极使用中文机构名称和文档类别。

常用搜索词：
- `研究所`
- `行业深度`
- `策略年度`
- `晨会纪要`
- `专题报告`
- `深度报告`

默认规则：
- 优先选择官方研究所或券商洞察页面
- 优先选择微信公众号作为券商研报首发渠道
- 如果市场主要暴露报告元数据而非 PDF，先返回元数据和官方路径

## 避免

- 暗示转载是授权的
- 省略访问限制的说明
- 在存在官方页面的情况下使用三级摘要
- 将公开思想领导力与分析师研究混淆而不标注区别
- 纳入需要付费或登录才能访问的研究
