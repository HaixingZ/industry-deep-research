# 来源优先级

除非用户给出更严格的指示，否则按以下顺序排列。

## 优先级阶梯

1. `official PDF`
   由出版机构、发行人、监管机构、交易所、公司、协会或授权活动页面直接托管的 PDF。
2. `official page`
   含有报告、演示文稿、会议记录或下载按钮的官方落地页，即使 PDF 本身不可用。
3. `wechat official account` — 微信公众号文章
   经认证的微信公众号发布的原创研究、行业分析、市场数据或官方公告。优先企业官方公众号、行业协会公众号、知名咨询公司公众号。
4. `authorized distribution`
   公开可访问的数据库、活动微型网站或合作伙伴页面，明确标注原始出版方和文档信息。
5. `secondary mirror`
   转载、聚合站或文件托管平台。仅在无法公开找到原始来源时使用，并标注风险。
6. `commentary`
   引用该报告的新闻报道、博客、演示文稿或 SEO 摘要。仅用于发现线索或补充背景，不作为主要交付物。

注意：遇到付费墙、登录墙或仅客户可访问的来源，直接跳过，不纳入 Source Index。

## 首选来源类别

### 1. 咨询和专业服务研究 (consulting report)

优先选择来自原始机构的命名研究中心、洞察页面和可下载报告。

典型目标：
- McKinsey, BCG, Bain
- Deloitte, PwC, EY
- Kearney, Oliver Wyman, Roland Berger
- Accenture, Capgemini

### 2. 监管机构、交易所和公共机构 (regulator/exchange)

当用户需要市场结构、政策、合规或官方统计数据时使用。

典型目标：
- 部委和政府机构
- 证券监管机构
- 证券交易所（上交所、深交所、北交所）
- 行业监管机构
- 统计机构
- 中央银行

### 3. 公司投资者关系与年度出版物 (company IR)

用于战略分析、市场规模线索、产品路线图、年报、投资者演示文稿、资本市场日材料和可持续发展报告。

典型目标：
- 投资者关系网站
- 年报门户
- 业绩演示资料库
- 投资者日页面

### 4. 标准机构、行业协会和产业联盟 (association/standard)

用于技术白皮书、标准、基准框架和全行业采用数据。

典型目标：
- 标准化组织
- 行业协会
- 提供可下载论文的会议论文集
- 协会研究中心

### 5. 微信公众号 (wechat official account)

搜索时通过搜狗微信搜索或 mp.weixin.qq.com 域名定位。优先选择：
- 经认证的企业/机构官方公众号
- 行业协会公众号
- 知名咨询公司公众号
- 券商研究所公众号
- 官方媒体公众号（如人民日报、新华社、经济观察报等）

标注为 `wechat official account`。

### 6. 券商和银行公开研究 (broker research)

仅纳入券商、银行官方研究平台上公开发布的研究成果。

优先搜索：
- 机构官方网站的洞察/研究页面
- 公开的年度展望、策略报告、行业深度
- 公开的会议演示材料、行业入门读本

保留原始标题、分析师或研究部门名称（如有）及日期。仅使用公开可访问的版本。付费墙后的研究直接跳过。

### 7. 中国互联网平台公开数据与研究

典型目标：
- 巨量引擎、腾讯广告等平台的公开营销研究报告
- 各平台研究院发布的公开行业报告
- 第三方数据平台（QuestMobile、艾瑞等）的公开摘要或报告

## 排名规则

以下情况得分更高：
- 更接近原始出版方
- 在相关时间窗口内更新
- 直接涉及请求主题而非相邻话题
- 可下载为 PDF 或演示文稿
- 有明确日期和可追溯的作者/机构

以下情况得分更低：
- 未标注日期
- 被第三方大量摘要
- 跨多个镜像站重复且无来源追溯
- 以 SEO 优化为目的而非出版质量

## 平分决胜规则

当两个来源看起来相似时，优先选择：
1. 原始托管站优于转载
2. PDF 优于 HTML 摘要
3. 有明确署名机构优于通用门户
4. 精确出版日期优于模糊的仅年份标注
5. 综合旗舰报告优于简短的衍生摘要
